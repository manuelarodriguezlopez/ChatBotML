¿Qué es el modelo de predicción?

El modelo de Machine Learning desarrollado por los estudiantes analiza la fruta base y la cantidad de producto a elaborar, recomendando automáticamente la técnica óptima de extracción de azúcares para maximizar la calidad del kéfir.
